# Мой публичный PGP-ключ
[Читать на английском](README.md)

Этот репозиторий содержит мой публичный PGP-ключ. Он используется для шифрования файлов и верификации моей цифровой подписи.

## Отпечаток ключа (Fingerprint)

Прежде чем импортировать ключ, **убедитесь, что его отпечаток совпадает**. Это гарантирует, что вы используете подлинный ключ.

Отпечаток: `F8D7 98CE 0285 8682 B383 E32C AB9A E9B8 F566 F71E`

## Как импортировать ключ
### С помощью командной строки (GnuPG)
 * Скачать ключ:

   `wget https://raw.githubusercontent.com/AukiCat/PGP-Keys/main/Auki_cat_public.asc`

 * Импортировать в вашу связку ключей:

   `gpg --import Auki_cat_public.asc`

 * Проверить отпечаток:

   `gpg --fingerprint AukiCat`
  
### С помощью Kleopatra (Windows)
 * Скачайте файл ключа `Auki_cat_public.asc` из этого репозитория.
 * Откройте Kleopatra.
 * Нажмите **Импорт**.
 * Выберите скачанный файл `Auki_cat_public.asc`.
 * Kleopatra импортирует ключ.
 * Чтобы проверить отпечаток, найдите импортированный ключ в списке, кликните по нему правой кнопкой мыши и выберите **Подробности**.
